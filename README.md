# image-api-using-codeigniter-elastic-search

## Features
- Upload Images to Server
- Hash file to avoid redundancy
- Add image file details to elastic
- A simple API call that serves the image

## Setup :
- PHP 5.5
- Composer
- Codeigniter 3.x
- Elastic
